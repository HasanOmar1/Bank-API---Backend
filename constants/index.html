Welcome

If you want to see all of the bank's data , go to this URL:

/api/v1/bank

If you want to see a specific user info , go to this URL:

/api/v1/bank/ [id of the user]

If you want to search for a user by name/letter , go to this URL:

/api/v1/bank/users/name?search= [letter/name]

If you want to search for all active users , go to this URL:

/api/v1/bank/active-users/true

If you want to search for all inActive users , go to this URL:

/api/v1/bank/active-users/false

If you want to search for all users who has higher/equal cash to x amount , go to this URL:

/api/v1/bank/users/higher-than?cash= [x amount]

If you want to search for all users who has lower/equal cash to x amount , go to this URL:

/api/v1/bank/users/lower-than?cash= [x amount]

If you want to search for active users who has higher/equal cash to x amount , go to this URL:

/api/v1/bank/active-users/true/higher-than?cash= [x amount]

If you want to search for active users who has lower/equal cash to x amount , go to this URL:

/api/v1/bank/active-users/true/lower-than?cash= [x amount]
